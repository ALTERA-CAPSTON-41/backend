## Getting Started

### Prerequisites

When you're going to contribute or compile, you'll need at least:
  - Go 1.18+
  - MySQL 8.0.x
  - MongoDB 5.x

### Installation or Configure

```bash
# clone if you don't have the code base
$ git clone git@github.com:ALTERA-CAPSTON-41/backend.git

# tidy up and run
$ go mod tidy

$ go run main.go
```
